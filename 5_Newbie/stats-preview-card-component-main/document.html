Stats Preview Card Component

Project Documentation & Technical Breakdown

Project Overview

This project is a solution to the Stats Preview Card Component challenge from Frontend Mentor. The goal was to build a highly responsive card component that displays business statistics with a specific design aesthetic, including a tinted image overlay.

Technologies Used

HTML5 CSS3 (Custom Properties) Flexbox CSS Grid Mobile-First Workflow

Key Technical Decisions

1. Layout Strategy

I used CSS Grid on the body to perfectly center the card on the viewport. For the card itself, I employed Flexbox with flex-wrap: wrap. This allowed for an easy transition from a vertical stack on mobile to a side-by-side layout on desktop using a media query at 768px.

2. Responsive Images

I implemented the <picture> element to serve different image assets based on the user's screen width. This ensures optimal performance and correct cropping for both mobile and desktop views.

<picture class="card-image">
    <source media="(min-width: 768px)" srcset="images/image-header-desktop.jpg">
    <img src="images/image-header-mobile.jpg" alt="Hero image">
</picture>

3. The Purple Overlay Effect

To achieve the specific purple tint seen in the design, I used a combination of background-color on the image container and mix-blend-mode: multiply on the image itself. This allows the purple color to "bleed" through the dark areas of the photo.

.card-image {
    background-color: var(--purple-500);
}

img {
    mix-blend-mode: multiply;
    opacity: 0.75;
}

4. Typography & Design Tokens

Following the style guide, I used CSS variables for colors and font families (Inter and Lexend Deca). This makes the code maintainable and ensures consistency across the component.

Responsive Design

The component is fully responsive:

  • Mobile: Stacked layout with centered text and stats.
  • Desktop: Side-by-side layout with left-aligned text and horizontal stats distribution.
  • Breakpoints: A single media query at 768px handles the transition smoothly.
